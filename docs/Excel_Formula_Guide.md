# 彩票控端盤口：Excel 精算公式與防呆架構指南

這份文件記錄了博弈彩票系統中，關於「機率、賠率、理論成本、利潤推算」的核心數學模型與 Excel 公式最佳實踐。這套標準能確保從「成本盤」到「現金盤」的所有反推數值，在小數點後第 N 位都能完美對齊，徹底消滅浮點數的四捨五入誤差。

---

## 🛑 核心數學鐵律 (The Golden Rule)

> [!IMPORTANT]
> **嚴禁使用「機率」作為任何金流與成本算式的分母或乘數！**
> 機率（例如 1/3 = 33.3333%）天生帶有無限循環小數，只要在 Excel 格式上被進位或截斷（例如顯示為 1.2755%），拿去乘上動輒上千倍的賠率後，就會引發極大的盤差誤差。
> 
> **唯一真理：** 整張大表的所有公式，**永遠只能使用「理論賠率（絕對有理數）」來做相除！**

---

## 1. 基礎基底：機率與理論賠率的「平行法則」

所有玩法的機率與理論賠率，都必須「各自獨立」從最原始的總數與中獎數推導出來。遇到有「和局」的玩法時，和局不貢獻損益，必須先將其從總數中拔除，得出**「有效總數」**。

**假設資料皆位在第 3 列：**
*   **D3** = 總組合數
*   **E3** = 主獎中獎次數
*   **H3** = 副獎中獎次數
*   **K3** = 和局次數

請將以下的公式貼入對應的欄位中（`IFERROR` 可防止沒有副獎或沒有和局時跳出 `#DIV/0!` 錯誤）：

*   **F 欄（主獎機率 %）**：`=IFERROR(E3 / (D3 - K3), "")`
*   **G 欄（主獎理論賠率）**：`=IFERROR((D3 - K3) / E3, "")`
*   **I 欄（副獎機率 %）**：`=IFERROR(H3 / (D3 - K3), "")`
*   **J 欄（副獎理論賠率）**：`=IFERROR((D3 - K3) / H3, "")`

> [!TIP]
> 貼上機率欄位（F、I）後，直接點擊 Excel 頂端的 `%` 符號來格式化顯示即可，**千萬不要套用 `=ROUND()` 函數**。

---

## 2. 理論成本的計算 (Base 100)

任何盤口的**理論成本**計算，都是要把該盤的 **`(主給定賠率 ÷ 主理論賠率) + (副給定賠率 ÷ 副理論賠率)`**，算完後因為該區塊的滿水成本是 100，所以整體需乘上 100。

### 依據你的 10 欄循環排列：
**【原始基準盤】** (L 主賠 / M 副賠)
*   **N 欄（理論成本）**：`=( IFERROR(L3/G3, 0) + IFERROR(M3/J3, 0) ) * 100`

**【開發數據 - 成本 A 盤】** (V 主賠 / W 副賠)
*   **X 欄（理論成本）**：`=( IFERROR(V3/G3, 0) + IFERROR(W3/J3, 0) ) * 100`

**【開發數據 - 成本 B 盤】** (AD 主賠 / AE 副賠)
*   **AF 欄（理論成本）**：`=( IFERROR(AD3/G3, 0) + IFERROR(AE3/J3, 0) ) * 100`

## 3. 信用利潤計算 (以百分比顯示)

在信用盤（有退水）的架構中，當前利潤等於 **`給定成本 - 理論成本`**。
為了讓 Excel 透過 `%` 按鈕漂亮地顯示（如 `0.1959%`），必須在相減後 **除以 100**。

**【原始基準盤】** (P 給定成本 / N 理論成本)
*   **Q 欄（信用利潤 %）**：`= (P3 - N3) / 100`

**【開發數據 - 成本 A 盤】** (Z 給定成本 / X 理論成本)
*   **AA 欄（信用利潤 %）**：`= (Z3 - X3) / 100`

---

## 4. 特殊連動反推：成本 B 盤的給定退水 (AG)

在你的設計中，**成本 B 盤** 負有「嚴格對齊利潤」的使命。
它的最終目標利潤落在 **AI 欄**（此欄通常 = AA 欄 + 手動微調值）。
既然利潤 (AI) 已被決定，我們就可以反向推導 B 盤的**給定退水 (AG)**：

`退水 = 100 - 給定成本`
`給定成本 = 理論成本(AF) + 目標利潤(AI × 100)`

*   **AG 欄（給定退水）**：`= 100 - (AF3 + (AI3 * 100))`

---

## 5. 現金盤架構：無退水的利潤與賠率反推

現金盤（Odds Markets）的特色是「滿水 100 買斷，沒有退水」，所以在數學上，它的給定成本永遠是 `1` (或 100%)。

### 算出現金利潤 % (T 欄)
如果在 R 到 T 的現金盤區塊，你已經填好了給定賠率：
*   **R** = 給定賠率
*   **S** = 給定副獎賠率
*   **T 欄（現金利潤 %）**：`= 1 - IFERROR(R3/G3, 0) - IFERROR(S3/J3, 0)`

### 反推給定賠率 (AK、AO)
如果現金盤的反推邏輯是「給定我一個死目標利潤，告訴我主獎該開多少賠率？」
以**賠率 A 盤**為例，假設它的基準現金利潤為 **AM 欄**：

> **終極連動版公式：**
> 讓現金盤的賠率能夠全自動吸納「成本 A 盤」被更改過後產生的利潤差額（也就是你說的 Delta: `當前利益 AA - 原始利益 AB`）。

*   **AK 欄（賠率 A 盤主給定賠率）**：
    `= G3 * ( 1 - (AM3 + (AA3 - AB3)) - IFERROR(AL3/J3, 0) )`

*   **AO 欄（賠率 B 盤主給定賠率，若其基準利潤為 AQ）**：
    `= G3 * ( 1 - (AQ3 + (AA3 - AB3)) - IFERROR(AP3/J3, 0) )`

> [!WARNING]
> 一個現金盤的主獎賠率（AK）與副獎賠率（AL），**只能選擇其中一個當作被公式反推的欄位**，另一個必須為手動輸入的常數，否則會在 Excel 引發「循環參照」的無限迴圈錯誤。
