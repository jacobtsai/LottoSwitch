## 重點業務邏輯已實現

1. **靜態 JSON 資料打包防護**

2. **Master-Slave 動態連動引擎**
   - 畫面分成 4 個盤口區塊。
   - 盤手專注於 **成本 A 盤 (Master)** 進行操作。
   - 一但輸入框變更（例如：修改給定退水），系統會在內部產生一組「利潤偏移量 (Delta Profit)」。
   - **成本 B 盤**與**現金 A/B 盤 (Slave)**，將瞬間同步捕捉這個 Delta，並完美精確地在各自維持其「基本利潤值」的前提下，動態反推算出「新退水」或「新現金賠率」。
   - 每一個盤口也都如實加上了「**利潤疊加（自訂溢價）**」欄位，供更細部的個別割草微調。

3. **High-End UI (Glassmorphism)**
   我們為介面套用了現代的太空灰底色、毛玻璃效果 (Backdrop Blur) 與科技感字體，將繁雜的數據呈現做得井然有序且賞心悅目。
